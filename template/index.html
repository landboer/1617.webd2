Wie zijn we?

I have a long ways to go before I am remotely good at web design. But I really like the thought of web design. It's all about creation. And that seems fun.

Wat bieden we aan?

I'm getting places with web development. I'm really good with the markup stuff. I'm getting better with JavaScript. I'm getting better with the frameworks and libraries that go along with JavaScript. But that's about all. I need to work smarter.

I think databases are a really important thing to learn. I don't know Node.js yet or SQL or anything relating to databases. So, I should probably get on that still. I have a long ways to go before I'm any good at any of this stuff, honestly.

My plans:

Right now, my main focuses are mostly JavaScript. I'm enrolled at Udacity.com, learning as much as possible about front-end development as I possibly can. It's going well, but I could do better.

I really want to become great at front-end design, but still know how to design full scale websites. Even if I don't become great at design, I still want to know how to design, and how to make something beautiful.

Design seems to be a whole different thought process. To program, you need to be able to think linearly, be able to break tasks down into tiny parts and complete those tasks. Design seems to be something different. You have to build something that lasts. And in the technology world, that is a challenge I would love to tackle.
